# Bubble
Bubble é um projeto individual criado por mim para estudo, e futuramente criar um jogo usando esse engine.
## Instalação
De preferência usando o vcpkg, instale as seguintes bibliotecas:
```bash
vcpkg install glfw3
vcpkg install glad
vcpkg install assimp
```
